About MathGem

MathGem is a website that generates printable basic math (arithmetic) worksheets for free. The website does not require any registration or login.

For users: inquiries, issues, and feature request

How To Print Worksheet?

Ctrl + P from the browser to print

It is already set to print the questions and answers properly, but it can be viewed/printed differently on each browser or on each computer. By default, the header (where the logo and top menus are located) will not be shown on the printed page.

Adjust size

On the Print pop-up screen, it is possible to change the print settings. If you want to include more questions per page, just change the scale to 70% or to any ratio you want.

Answer sheet

The answer sheet will be printed at the end of the pages.
