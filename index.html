About MathGem

MathGem is a website that provides printable math (arithmetic) worksheets for free. We do not require any registration or login.

For users: inquiries, issues, and feature request

Hi, I am Thomas. Thanks for visiting MathGem.ca. I initially built this web page for my kids and for my wife. :) I didn't spend too much time on it, so it might have some issues. If you find any problems or if you want any additional features, please contact me by email leez79@gmail.com.

How To Print Worksheet?

Ctrl + P from the browser to print

It is already set to print the questions and the answer sheet properly, but I tested this only on Chrome. It can be viewed or printed differently on different browsers. By default, the control panel will not be printed. The answer sheet is not visible on browser, but it will be printed after the questions.

Adjust size

On the Print pop-up screen, it is possible to change the print settings. If you want to include more questions per page, just change the scale to 70% or to any ratio you want.
